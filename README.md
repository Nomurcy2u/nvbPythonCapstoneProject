It takes and analyzes the nvb_cybersecurity_incidents.csv file and returns graphs and insights specifically regarding the data provided.

2. Upload the dataset file if required (Option A projects)
3. Run all cells from top to bottom (There is only 1 cell)

- Dataframes
- Graph Plotting (Through matplotlib)
- groupby analysis
